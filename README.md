# — Irmax (aka owermessage)

### 1 — О проекте:
Данное приложение для передачи сообщений в реальном времени создан. Это протоколу web—socket благодаря выполняется, в реальном времени сообщения передаются.
Для учебной практики мессенджер предназначен. Сайт с друзьями, родными, знакомыми, незнакомыми, братьями, сестрами, сватами, и т.д общаться позволяет.

### 2 — Fast start
Что — бы приложение для обмена сообщениями запустить, несколько шагов сделать требуется:
1. Сервер (виртуальный, выделенный) арендуйте, либо белый IP — Адрес у провайдера своего арендуйте.
2. По SSH на свой сервер войдите, далее git clone https://github.com/qqkane/irmax пропишите.
3. Через Docker контейнер будем запускать.
4. nano Dockerfile напечатайте, и данный текст туда введите:
```
FROM python:3.10-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
```
5. Далее введите nano docker-compose.yml, и данный текст перенесите в сие файловое пространство:
```
version: '3.11'

services:
  app:
    build: .
    ports:
      - "8000:8000"
    volumes:
      - ./messenger.db:/app/messenger.db
    restart: always
```

6. Слова заморские "docker-compose.yml up -d --build" напишите.
7. Браузер откройте, и по ссылке https://IP:Порт (8000) перейдите.
8. Учетную запись в сообщение-обменнике пароль и логин введя создайте.
9. По любому имени слева нажмите, и глаголить начинайте.

### 3 — Кода объяснение

Проект на FastAPI в основном написан, логика его на нем работает, база данных — SQLite эксплуатируется.
main.py — за роуты, вебсокеты, логику чата отвечает.
models.py — таблицы: юзеры, сообщения, токены.
schemas.py — валидация данных через pydantic.
database.py — подключение к БД.

Пароли через sha256 ради безопасности шифруются, авторизация по токену.
